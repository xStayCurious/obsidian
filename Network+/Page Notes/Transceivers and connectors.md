-----------------------------------
TRANSCEIVERS
-----------------------------------

Transceiver =
device that:
- transmits
and:
- receives signals

The guide references:
- transceivers used with fiber and network media

-----------------------------------
CORE PURPOSE
-----------------------------------

Transceivers convert:
- electrical signals
and/or:
- optical signals

to support communication between devices.

-----------------------------------
COMMON TRANSCEIVER USES
-----------------------------------

Used in:
- switches
- routers
- fiber networking
- network uplinks

-----------------------------------
CONNECTORS
-----------------------------------

Connectors are used to:
- physically connect network media
to:
- devices and interfaces

-----------------------------------
SC CONNECTOR
-----------------------------------

SC =
Subscriber Connector

Characteristics:
- square-shaped
- push-pull connector design

Used with:
- fiber optic cable

-----------------------------------
SC PURPOSE
-----------------------------------

SC connectors are commonly used in:
- fiber optic networking
- enterprise fiber installations

-----------------------------------
LC CONNECTOR
-----------------------------------

LC =
Lucent Connector

Characteristics:
- small form-factor connector
- compact size

Used with:
- fiber optic cable

-----------------------------------
LC PURPOSE
-----------------------------------

LC connectors are commonly used in:
- modern fiber networking
- high-density fiber environments

-----------------------------------
ST CONNECTOR
-----------------------------------

ST =
Straight Tip

Characteristics:
- twist-lock mechanism

Used with:
- fiber optic cable

-----------------------------------
ST PURPOSE
-----------------------------------

ST connectors are commonly found in:
- older fiber installations

-----------------------------------

-----------------------------------
RJ11 CONNECTOR
-----------------------------------

RJ11:
- connector used for telephone lines

Characteristics:
- smaller than RJ45
- fewer pins

-----------------------------------
RJ11 USES
-----------------------------------

Used for:
- telephone connections
- DSL connections

-----------------------------------
RJ45 CONNECTOR
-----------------------------------

RJ45:
- standard connector for Ethernet networking

Used with:
- twisted pair Ethernet cable

-----------------------------------
RJ45 CHARACTERISTICS
-----------------------------------

- larger than RJ11
- supports Ethernet communication
- commonly uses 8 pins

-----------------------------------
RJ45 USES
-----------------------------------

Used for:
- Ethernet networking
- LAN connections
- switches and routers

-----------------------------------
F-TYPE CONNECTOR
-----------------------------------

F-type connector:
- used with coaxial cable

-----------------------------------
F-TYPE USES
-----------------------------------

Commonly used for:
- cable television
- cable internet
- modems

-----------------------------------
BNC CONNECTOR
-----------------------------------

BNC =
Bayonet Neill-Concelman

Characteristics:
- twist-lock/bayonet-style connector

Used with:
- coaxial cable

-----------------------------------
BNC USES
-----------------------------------

Commonly used for:
- older networking technologies
- video systems
- radio frequency applications

-----------------------------------
FIBER CONNECTOR COMPARISON
-----------------------------------

SC:
- square push-pull design

LC:
- smaller compact connector

ST:
- twist-lock connector

MPO:
- multiple fibers in one connector

-----------------------------------
COPPER CONNECTOR COMPARISON
-----------------------------------

RJ11:
- telephone/DSL

RJ45:
- Ethernet networking

-----------------------------------
COAXIAL CONNECTOR COMPARISON
-----------------------------------

F-type:
- cable TV/internet

-----------------------------------
COMMON EXAM TRAPS
-----------------------------------

Trap:
Confusing RJ11 and RJ45.

RJ11:
- telephone/DSL

RJ45:
- Ethernet

-----------------------------------

Trap:
Thinking BNC is a fiber connector.

Wrong.

BNC:
- coaxial connector

-----------------------------------

Trap:
Confusing SC and LC.

SC:
- larger square connector

LC:
- smaller compact connector

-----------------------------------

Trap:
Thinking MPO carries only one fiber.

Wrong.

MPO:
- supports multiple fibers
