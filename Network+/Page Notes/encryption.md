WIRELESS ENCRYPTION
-----------------------------------

Wireless encryption protects:
- wireless data confidentiality
- wireless communications
- transmitted information

-----------------------------------
CORE PURPOSE
-----------------------------------

Encryption prevents:
- unauthorized reading of wireless traffic

Without encryption:
- nearby attackers may intercept wireless traffic

-----------------------------------
WHY WIRELESS ENCRYPTION MATTERS
-----------------------------------

Wireless traffic:
- travels through the air
- can potentially be intercepted by nearby devices

Encryption protects:
- usernames
- passwords
- private data
- business communications

-----------------------------------
MAIN EXAM ENCRYPTION TERMS
-----------------------------------

The major Network+ wireless encryption/security terms:
- WEP
- WPA
- WPA2
- WPA3
- TKIP
- AES

-----------------------------------
WEP
-----------------------------------

WEP =
Wired Equivalent Privacy

-----------------------------------
WEP PURPOSE
-----------------------------------

WEP was:
- an early wireless encryption standard

-----------------------------------
WEP CHARACTERISTICS
-----------------------------------

- very old
- insecure
- deprecated
- vulnerable to attack

-----------------------------------
WEP EXAM IDEA
-----------------------------------

If the exam asks:
"What wireless encryption is weakest or deprecated?"

Answer:
WEP

-----------------------------------
WHY WEP IS INSECURE
-----------------------------------

WEP contains:
- major cryptographic weaknesses
- weak key management
- predictable vulnerabilities

-----------------------------------
WEP EXAM KEYWORDS
-----------------------------------

WEP exam keywords:
- weak
- deprecated
- insecure
- legacy wireless security

-----------------------------------
WPA
-----------------------------------

WPA =
Wi-Fi Protected Access

-----------------------------------
WPA PURPOSE
-----------------------------------

WPA was created to:
- improve wireless security
- replace WEP

-----------------------------------
WPA CHARACTERISTICS
-----------------------------------

- transitional improvement over WEP
- stronger than WEP
- older than WPA2/WPA3

-----------------------------------
TKIP
-----------------------------------

TKIP =
Temporal Key Integrity Protocol

-----------------------------------
TKIP PURPOSE
-----------------------------------

TKIP was introduced with:
- WPA

Purpose:
- improve encryption security over WEP

-----------------------------------
TKIP EXAM IDEA
-----------------------------------

TKIP is considered:
- older
- weaker than AES
- legacy technology

-----------------------------------
WPA2
-----------------------------------

WPA2:
- significantly improved wireless security
- became the standard enterprise/home WiFi security method for years

-----------------------------------
AES
-----------------------------------

AES =
Advanced Encryption Standard

-----------------------------------
WPA2 AND AES
-----------------------------------

Very important exam fact:

WPA2 is strongly associated with:
- AES

-----------------------------------
AES CHARACTERISTICS
-----------------------------------

AES is:
- stronger
- modern
- more secure
than:
- TKIP

-----------------------------------
WPA2 EXAM IDEA
-----------------------------------

If asked:
"What encryption does WPA2 use?"

Answer:
AES

-----------------------------------
WPA3
-----------------------------------

WPA3:
- newer than WPA2
- stronger than WPA2
- modern wireless security standard

-----------------------------------
WPA3 IMPROVEMENTS
-----------------------------------

WPA3 improves:
- authentication
- encryption strength
- password protection

-----------------------------------
WPA3 EXAM IDEA
-----------------------------------

If asked:
"What is the strongest modern wireless encryption/security standard?"

Answer:
WPA3

-----------------------------------
WIRELESS SECURITY ORDER
-----------------------------------

From weakest to strongest:

WEP
WPA
WPA2
WPA3

-----------------------------------
PSK
-----------------------------------

PSK =
Pre-Shared Key

-----------------------------------
PSK PURPOSE
-----------------------------------

PSK uses:
- a shared wireless password/passphrase

-----------------------------------
PSK CHARACTERISTICS
-----------------------------------

- common in homes
- simple configuration
- all users share same password

-----------------------------------
PSK WEAKNESSES
-----------------------------------

Weaknesses:
- password sharing
- difficult user accountability
- harder credential management

-----------------------------------
ENTERPRISE AUTHENTICATION
-----------------------------------

Enterprise wireless authentication:
- uses centralized authentication
- often uses RADIUS servers
- authenticates users individually

-----------------------------------
RADIUS
-----------------------------------

RADIUS provides:
- centralized authentication
- individual user validation
- enterprise wireless access control

WPA PERSONAL VS ENTERPRISE
-----------------------------------

Personal mode:
- shared password
- PSK-based

Enterprise mode:
- centralized authentication
- individual credentials
- RADIUS/802.1X

-----------------------------------
OPEN NETWORKS
-----------------------------------

Open wireless networks:
- use no encryption
- use no authentication

-----------------------------------
OPEN NETWORK RISKS
-----------------------------------

Open networks are vulnerable to:
- eavesdropping
- traffic interception
- man-in-the-middle attacks
- unauthorized access

-----------------------------------
CAPTIVE PORTALS
-----------------------------------

Guest networks may use:
- captive portals

Captive portals may require:
- login
- agreement acceptance
- payment information

before granting internet access.

-----------------------------------
ENCRYPTION VS AUTHENTICATION
-----------------------------------

Encryption:
- protects confidentiality of data

Authentication:
- verifies identity

HIGH-YIELD NETWORK+ FACTS
-----------------------------------

Very important exam facts:

- WEP is insecure/deprecated
- WPA replaced WEP
- WPA2 commonly uses AES
- TKIP is older/weaker than AES
- WPA3 is strongest/common modern standard
- PSK uses shared passwords
- Enterprise authentication uses RADIUS/802.1X
- Open networks provide no encryption
- Encryption protects confidentiality
- Authentication verifies identity

COMMON EXAM TRAPS
-----------------------------------

Trap:
Thinking WEP is still secure.

Wrong.

WEP:
- is broken/insecure

-----------------------------------

Trap:
Confusing AES and TKIP.

AES:
- stronger
- modern

TKIP:
- older
- weaker

-----------------------------------

Trap:
Thinking WPA2 and WPA3 are encryption algorithms themselves.

Wrong.

They are:
- wireless security standards/frameworks

AES is the encryption algorithm.

-----------------------------------

Trap:
Thinking PSK means each user has unique credentials.

Wrong.

PSK:
- shared password for all users

-----------------------------------

Trap:
Thinking open networks are encrypted.

Wrong.

Open networks:
- provide no encryption

-----------------------------------

Trap:
Confusing authentication and encryption.

Authentication:
- verifies identity

Encryption:
- protects confidentiality
