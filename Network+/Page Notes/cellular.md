-----------------------------------
CELLULAR TECHNOLOGY
-----------------------------------

Cellular technology is:
- a wireless communication method
that:
- uses a network of cell sites

Each site covers:
- a specific area called a cell

-----------------------------------
CORE PURPOSE
-----------------------------------

Cellular networks provide:
- wireless communication
- mobile connectivity
- internet access
- voice communication

-----------------------------------
KEY CHARACTERISTIC
-----------------------------------

The fundamental feature of cellular networks is:
- frequency reuse

-----------------------------------
FREQUENCY REUSE
-----------------------------------

Frequency reuse means:
- the same frequencies can be reused
in different cells
to:
- increase network capacity
- increase coverage

-----------------------------------
CELL STRUCTURE
-----------------------------------

A cellular network is divided into:
- cells

Each cell is served by:
- a cell site/tower

-----------------------------------
WHY CELLS EXIST
-----------------------------------

Cells help:
- organize wireless coverage
- improve scalability
- support many users simultaneously

-----------------------------------
CELLULAR GENERATIONS
-----------------------------------

Modern cellular networks are divided into:
- 2G
- 3G
- 4G
- 5G

-----------------------------------
2G
-----------------------------------

2G:
- second generation cellular
- focused primarily on digital voice communication

-----------------------------------
3G
-----------------------------------

3G:
- improved mobile data capabilities
- supported internet access and multimedia services

-----------------------------------
4G
-----------------------------------

4G:
- higher-speed mobile networking
- improved streaming and internet performance

-----------------------------------
5G
-----------------------------------

5G:
- newest generation listed in the guide
- increased data speeds
- improved connectivity features

-----------------------------------
CELLULAR CAPABILITIES
-----------------------------------

Cellular technology supports:
- voice calls
- mobile internet access
- video streaming
- IoT connectivity

-----------------------------------
MOBILE INTERNET ACCESS
-----------------------------------

Cellular networks allow devices to:
- access the internet wirelessly
while:
- moving between locations

-----------------------------------
VIDEO STREAMING
-----------------------------------

Modern cellular networks support:
- high-bandwidth applications
such as:
- video streaming

-----------------------------------
IOT CONNECTIVITY
-----------------------------------

Cellular technology supports:
- Internet of Things (IoT) devices

Examples:
- smart sensors
- connected devices
- remote monitoring systems

-----------------------------------
WIRELESS COMMUNICATION
-----------------------------------

Cellular communication:
- does not require physical cabling
- uses radio frequencies

-----------------------------------
COVERAGE
-----------------------------------

Cellular networks are designed to provide:
- broad geographic coverage
- mobile communication services

-----------------------------------
SCALABILITY
-----------------------------------

Frequency reuse and cell division help:
- support many users
- improve network scalability
